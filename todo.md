# Development Roadmap: Echoes & Visions Museum

## Navigation & Movement
- [ ] WASD + arrow key movement
- [ ] Click/tap to teleport
- [ ] Right-click drag to rotate camera
- [ ] Speed tiers (1/2/3 keys: Slow/Normal/Fast)
- [ ] Waypoint system — teleport to any room from minimap or atrium
- [ ] Head-bob animation while walking

## Painting Interactions
- [ ] Approach painting → scales up, room darkens slightly
- [ ] Info panel on click (artist, date, medium, dimensions)
- [ ] Analysis card (one-line critical insight)
- [ ] Deep zoom (high-res zoom to brushstroke level, press E)
- [ ] Favorite/heart a painting
- [ ] "Story" mode — 30s narrated text or TTS per painting

## Music Interactions
- [ ] Play/pause per music piece
- [ ] Waveform visualizer (real-time frequency bars in room)
- [ ] Score follower (simplified scrolling notation)
- [ ] Composer bio panel on portrait click
- [ ] Instrument spotlight — click instrument to hear it isolated

## Audio System
- [ ] Ambient music per wing (auto-switches by room)
- [ ] Audio companion — era-matched music plays near paintings
- [ ] Footstep sounds that change by floor material (marble, wood, stone)
- [ ] Spatial audio / reverb matching room size
- [ ] Mute/unmute toggle in HUD

## Special Modes
- [ ] Caravaggio Spotlight — dynamic chiaroscuro lighting near his paintings
- [ ] "Artist's Eye" toggle — annotated light paths overlay
- [ ] "Deaf Beethoven Mode" — muffled audio simulating hearing loss
- [ ] "Inside the Fugue" — Bach voices as colored 3D ribbons
- [ ] "Rain Room" — rain synced to Chopin's music, candles flicker

## HUD & UI
- [ ] Minimap (corner, always visible, visited rooms glow)
- [ ] Music toggle button
- [ ] Controls help overlay
- [ ] FPS counter (optional)
- [ ] Accessibility options
- [ ] Mobile touch controls

## Onboarding & Tutorial
- [ ] Start menu / landing screen
- [ ] Auto-walk through the door on first entry
- [ ] Tutorial text overlays explaining controls (WASD, click, E for zoom)
- [ ] Globe in lobby — hover pins for artist names, click for mini bio
- [ ] Floor mosaic timeline with artist icons

## Social & Persistence
- [ ] Favorites system (heart paintings + music, collected in atrium)
- [ ] Guestbook — type a reflection, save/share
- [ ] Share card — generate image with favorites + visit stats
- [ ] "Visit Again" — lobby with all rooms unlocked for free roam

## Immersive Chamber (Generative Experience)
- [ ] Auto-playing 3-min audiovisual experience
- [ ] Paintings dissolve/reform on walls, synced to music
- [ ] Procedurally unique each visit
- [ ] Skip button

## Performance & Loading
- [ ] Progressive texture loading (LOD: 4096 → 512 based on distance)
- [ ] Streaming audio (not preloaded)
- [ ] Loading screen with progress indicator
- [ ] Dynamic quality reduction when FPS drops
